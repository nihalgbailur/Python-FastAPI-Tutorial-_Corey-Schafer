## Large File Policy

- Use normal git for binaries in this repository for now.
- Prefer text-first learning resources (`.md`) when possible for readable diffs.
- Review before committing very large binary files (recommended threshold: files larger than 25 MB).
- If binary volume grows significantly, migrate future large binaries to Git LFS.
